# startTime

## Summary

The point in time when an element or an action starts,
becomes applicable, or becomes valid.

## Description

A timestamp marking the commencement of an action,
the beginning of an element's relevance,
or the intended start of either.

It encompasses the moment of triggering processes,
such as build executions or service instantiations,
as well as the point in time when an element
or a relationship between elements becomes effective or active.
