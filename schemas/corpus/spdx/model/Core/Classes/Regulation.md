# Regulation

## Summary

Regulation represents a rule or directive maintained by an authority.

## Description

The Regulation class represents any rules or directives enforced by an official authority.
Regulations within this class can be either mandatory or voluntary.

Examples of these regulations include the EU Radio Equipment Directive, EU Cyber Resilience Act,
the Japan Cyber STAR, or the U.S. Cyber Trust Mark.
